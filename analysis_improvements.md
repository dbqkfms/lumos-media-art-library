# LUMOS 웹사이트 분석 및 개선사항

## 📋 현재 상태 분석

### ✅ 구현된 기능
1. **디자인 시스템**: Luminous Brutalism 철학 적용
2. **Hero 섹션**: 풀스크린 애니메이션 배경 (이미지)
3. **이원화 구조**: STANDARD/LOCAL 50/50 분할
4. **갤러리 페이지**: STANDARD(Masonry), LOCAL(Grid)
5. **기본 네비게이션**: Back 버튼

---

## 🚨 **기능적으로 빠진 요소**

### 1. **영상 콘텐츠 부족** ⭐⭐⭐
**문제**: 라이브러리인데 영상이 거의 없음
- Hero 섹션: 이미지만 있음 (영상 플레이스홀더)
- 갤러리: 모두 정적 이미지만 표시
- 영상 미리보기 기능 없음

**해결 방안**:
- [ ] Hero 섹션에 실제 영상 추가
- [ ] "Video Gallery" 섹션 추가 (홈페이지)
- [ ] 갤러리에 영상 썸네일 + 호버 시 영상 재생
- [ ] 영상 모달 팝업 기능

### 2. **페이지 간 이동 불편** ⭐⭐
**문제**: STANDARD ↔ LOCAL 직접 이동 불가
- 각 페이지에서 홈으로만 돌아갈 수 있음
- 두 World 간 빠른 전환 불가능

**해결 방안**:
- [ ] 헤더에 "Switch to LOCAL/STANDARD" 버튼 추가
- [ ] 고정 네비게이션 바 (Home, STANDARD, LOCAL)

### 3. **폰트 시스템 미흡** ⭐⭐
**문제**: 폰트 선택이 명확하지 않음
- `font-display`, `font-accent` 사용하지만 실제 폰트 정의 없음
- Google Fonts 주석 처리되어 있음

**해결 방안**:
- [ ] Display 폰트: Playfair Display / Cormorant Garamond (우아함)
- [ ] Body 폰트: Inter / Work Sans (가독성)
- [ ] Accent 폰트: Space Grotesk / Archivo (기술적 느낌)

### 4. **글씨 색상 가독성 문제** ⭐⭐⭐
**문제**: 배경과 텍스트 대비 부족한 영역
- Home > World Selection: 어두운 배경 + 회색 텍스트
- About 섹션: 반투명 배경 + 회색 텍스트
- LOCAL 페이지: 밝은 배경인데 일부 텍스트 대비 약함

**해결 방안**:
- [ ] 텍스트 배경에 반투명 박스 추가
- [ ] 텍스트 그림자 효과 강화
- [ ] 색상 대비 검증 (WCAG AA 기준)

### 5. **Contact 기능 없음** ⭐
**문제**: "Contact Us" 버튼이 작동하지 않음
- 문의 폼 없음
- 이메일/전화번호 정보 없음

**해결 방안**:
- [ ] 문의 폼 페이지 추가
- [ ] Footer에 연락처 정보 추가
- [ ] 또는 간단한 모달 폼

### 6. **반응형 디자인 미흡** ⭐
**문제**: 모바일 최적화 부족
- World Selection: 모바일에서 세로 스택
- 갤러리: 모바일에서 간격 조정 필요

**해결 방안**:
- [ ] 모바일 네비게이션 메뉴
- [ ] 터치 제스처 최적화
- [ ] 모바일 폰트 크기 조정

### 7. **SEO 및 메타데이터 부족** ⭐
**문제**: 검색 엔진 최적화 없음
- 메타 태그 없음
- Open Graph 이미지 없음
- sitemap.xml 없음

**해결 방안**:
- [ ] 메타 태그 추가
- [ ] OG 이미지 설정
- [ ] robots.txt, sitemap.xml
