# 네비게이션 UI/UX 패턴 분석

## 📹 사용자 제공 레퍼런스 영상 분석 (삼성 웹사이트)

### 핵심 패턴: **Mega Menu (메가 메뉴)**

#### 구조
```
[메인 메뉴 1] [메인 메뉴 2] [메인 메뉴 3] ...
     ↓ (마우스 호버)
┌─────────────────────────────────────────┐
│  서브메뉴 1  서브메뉴 2  서브메뉴 3      │
│  서브메뉴 4  서브메뉴 5  서브메뉴 6      │
│                                          │
│  [제품 이미지 그리드]                     │
└─────────────────────────────────────────┘
```

#### 특징
1. **호버 시 드롭다운**: 메인 메뉴에 마우스를 올리면 큰 패널이 아래로 펼쳐짐
2. **세부 카테고리**: 한두 단어로 된 간결한 서브메뉴 (예: "갤럭시북", "데스크탑", "모니터")
3. **비주얼 그리드**: 서브메뉴 아래 제품 이미지 그리드 표시
4. **빠른 접근**: 모든 서브 카테고리에 한 번에 접근 가능
5. **우측 사이드바**: 추가 세부 옵션 (예: "더 알아보기", "Windows 11")

## 📐 상세 UI/UX 설계

### 1. 헤더 네비게이션 구조

```
┌──────────────────────────────────────────────────────────────┐
│ [LUMOS]  [STANDARD ▼]  [LOCAL ▼]  [About]  [Contact]  [🔍]  │
└──────────────────────────────────────────────────────────────┘
```

**인터랙션**:
- STANDARD/LOCAL에 마우스 호버 → 메가 메뉴 펼쳐짐
- 클릭 시 → 해당 페이지로 이동 (All 카테고리)
- 메가 메뉴에서 카테고리 클릭 → 해당 페이지의 필터링된 뷰로 이동

## 🎨 UI/UX 레퍼런스 사이트 분석

- **패턴**: Mega Menu with Product Grid
- **특징**: 
  - 호버 시 큰 드롭다운 패널
  - 제품 이미지 + 간결한 텍스트
  - 부드러운 애니메이션

### 2. **Nike.com**
- **패턴**: Mega Menu with Categories
- **특징**:
  - 좌측: 카테고리 리스트
  - 우측: 제품 이미지 그리드
  - 빠른 필터링

- **패턴**: Top Nav + Filter Tabs
- **특징**:
  - 상단: 메인 네비게이션
  - 하단: 카테고리 필터 탭
  - 무한 스크롤 갤러리

- **패턴**: Minimal Nav + Tag Filters
- **특징**:
  - 심플한 상단 네비게이션
  - 태그 기반 필터링
  - 그리드 + 리스트 뷰 전환

## 🎯 핵심 UX 원칙

1. **자유로운 이동**: STANDARD ↔ LOCAL 언제든지 전환 가능
2. **빠른 탐색**: 메가 메뉴로 모든 카테고리 한눈에 보기
3. **명확한 구분**: STANDARD(프리미엄) vs LOCAL(상업용) 시각적 차별화
4. **일관성**: 모든 페이지에서 동일한 네비게이션 패턴
5. **접근성**: 키보드 네비게이션, 포커스 상태 명확히

## 🎨 디자인 가이드

### 메가 메뉴 스타일
- **배경**: 반투명 블랙 (rgba(0,0,0,0.95))
- **애니메이션**: 0.3s ease-out (아래로 슬라이드)
- **그림자**: 부드러운 그림자 효과
- **간격**: 여유로운 패딩 (py-8, px-12)

### 카테고리 탭 스타일
- **기본**: text-gray-400
- **호버**: text-white (STANDARD), text-gray-800 (LOCAL)
- **활성**: border-bottom 골드 라인 (STANDARD), 블랙 라인 (LOCAL)
- **간격**: mx-4

### 작품 썸네일 스타일
- **크기**: 200px × 150px (메가 메뉴), 300px × 225px (갤러리)
- **호버**: scale(1.05), 그림자 강화
- **정보**: 제목 오버레이 (하단)
