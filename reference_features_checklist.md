# 레퍼런스 사이트 종합 분석 및 LUMOS 필요 기능 체크리스트

## 📊 분석 대상 사이트
1. **Niio.com** - 아트 스트리밍 플랫폼

## 🎯 **핵심 발견: LUMOS에 꼭 필요한 기능**

### 1. **콘텐츠 갤러리 탭/필터 시스템** ⭐⭐⭐ (최우선)

#### 현재 문제
- STANDARD/LOCAL 페이지에 각각 5개, 3개 작품만 표시
- 카테고리별 필터 없음
- 더 많은 콘텐츠를 볼 방법이 없음

#### 레퍼런스 사이트 패턴
**Niio.com**:
- "All", "Featured", "New", "Popular" 탭
- 카테고리 필터: Abstract, Nature, Geometric, etc.
- 검색 기능

**LED.art**:
- "All Works", "Collections", "Artists" 탭
- 스타일 필터: Minimalist, Bold, Organic
- 공간 타입 필터: Hotel, Office, Public

**Vido.gallery**:
- 카테고리 탭: Media Art, Installation, Performance
- 태그 필터 시스템
- 정렬 옵션: Latest, Popular, Trending

#### LUMOS 구현 계획
```
STANDARD 페이지:
┌─────────────────────────────────────────┐
│ [All] [Abstract] [Cosmic] [Material]    │ ← 탭
│ [Light] [Pattern]                        │
├─────────────────────────────────────────┤
│ 🎨 🎨 🎨 🎨 🎨 🎨 🎨 🎨 🎨 🎨 🎨 🎨    │ ← 갤러리 그리드
│ 🎨 🎨 🎨 🎨 🎨 🎨 🎨 🎨 🎨 🎨 🎨 🎨    │   (15-20개 작품)
└─────────────────────────────────────────┘

LOCAL 페이지:
┌─────────────────────────────────────────┐
│ [All] [Nature] [Seasonal] [Urban]       │ ← 탭
├─────────────────────────────────────────┤
│ 🌿 🌿 🌿 🌿 🌿 🌿 🌿 🌿 🌿 🌿 🌿 🌿    │ ← 갤러리 그리드
│ 🌿 🌿 🌿 🌿 🌿 🌿 🌿 🌿 🌿 🌿 🌿 🌿    │   (15-20개 작품)
└─────────────────────────────────────────┘
```

**구현 요소**:
- [ ] 카테고리 탭 버튼 (All, Abstract, Cosmic, Material, Light, Pattern)
- [ ] 탭 클릭 시 필터링 로직
- [ ] 갤러리 아이템 15-20개로 확장 (임시 이미지 생성)
- [ ] 호버 시 작품 정보 표시
- [ ] 반응형 그리드 (모바일: 1열, 태블릿: 2열, 데스크탑: 3열)

### 2. **사례(Case Studies) 섹션** ⭐⭐⭐

#### 레퍼런스 사이트 패턴
**Niio.com**:
- "Success Stories" 섹션
- 호텔, 오피스, 레스토랑 사례
- Before/After 이미지
- 고객 인터뷰 인용문

**LED.art**:
- "Featured Projects" 섹션
- 럭셔리 호텔, 공항, 거리 설치 사례
- 프로젝트 상세 페이지 (공간 사진, 작품 설명, 효과)

**Vido.gallery**:
- "Exhibitions" 섹션
- 전시 공간 사진 + 작품 설치 샷
- 날짜, 장소, 큐레이터 정보

#### LUMOS 구현 계획
```
홈페이지에 "Case Studies" 섹션 추가:
┌─────────────────────────────────────────┐
│           Case Studies                   │
│                                          │
│ ┌──────────┐ ┌──────────┐ ┌──────────┐ │
│ │ 호텔 로비 │ │ 갤러리   │ │ 카페     │ │
│ │ [이미지]  │ │ [이미지]  │ │ [이미지]  │ │
│ │ 설명...   │ │ 설명...   │ │ 설명...   │ │
│ └──────────┘ └──────────┘ └──────────┘ │
└─────────────────────────────────────────┘
```

**구현 요소**:
- [ ] Home 페이지에 Case Studies 섹션 추가
- [ ] 3-4개 사례 카드 (임시 이미지 + 설명)
- [ ] 사례 타입: 호텔, 갤러리, 카페, 오피스
- [ ] 각 사례: 공간 이미지 + 프로젝트명 + 간단한 설명
- [ ] 호버 시 "View Details" 버튼 (현재는 플레이스홀더)

**임시 사례 예시**:
1. **서울 호텔 로비** - STANDARD 작품으로 럭셔리 분위기 연출
2. **강남 갤러리** - 추상 미디어아트 전시
3. **홍대 카페** - LOCAL 자연 콘텐츠로 편안한 공간 조성
4. **판교 오피스** - 계절 콘텐츠로 업무 공간 활성화

### 3. **네비게이션 개선** ⭐⭐

#### 레퍼런스 사이트 패턴
**공통점**:
- 고정 헤더 네비게이션 (스크롤 시에도 항상 보임)
- 명확한 섹션 구분: Home, Gallery, About, Contact
- 페이지 간 빠른 전환 가능

**Niio.com**:
```
[NIIO] [Artworks] [Channels] [Pricing] [About] [Sign In]
```

**LED.art**:
```
[LED.ART] [Works] [Collections] [About] [Contact]
```

#### LUMOS 구현 계획
```
현재:
[← Back]                    [STANDARD]                    [ ]

개선:
[LUMOS] [Home] [STANDARD] [LOCAL] [About] [Contact]  [→ LOCAL]
                                                       ↑ 빠른 전환
```

**구현 요소**:
- [ ] 고정 헤더 네비게이션 (모든 페이지 공통)
- [ ] Home, STANDARD, LOCAL, About, Contact 링크
- [ ] STANDARD 페이지: "Switch to LOCAL →" 버튼
- [ ] LOCAL 페이지: "Switch to STANDARD →" 버튼
- [ ] 현재 페이지 하이라이트 표시

### 4. **작품 상세 모달/페이지** ⭐⭐

#### 레퍼런스 사이트 패턴
**공통점**:
- 갤러리 아이템 클릭 시 상세 정보 표시
- 큰 이미지/영상 프리뷰
- 작품 정보: 제목, 설명, 태그, 스펙
- "문의하기" CTA

**Niio.com**:
- 모달 팝업 형식
- 영상 전체화면 재생
- 아티스트 정보, 작품 설명
- "Add to Channel" 버튼

**LED.art**:
- 별도 상세 페이지
- 고화질 이미지 갤러리
- 작품 스펙 (해상도, 길이, 포맷)
- "Request License" 버튼

#### LUMOS 구현 계획
```
갤러리 아이템 클릭 시:
┌─────────────────────────────────────────┐
│                    ✕                     │
│  ┌───────────────────────────────────┐  │
│  │                                   │  │
│  │      [큰 이미지/영상 프리뷰]       │  │
│  │                                   │  │
│  └───────────────────────────────────┘  │
│                                          │
│  Cosmic Flow                             │
│  Abstract | 8K | 10s Loop               │
│                                          │
│  우주의 에너지가 흐르는 추상적 시각화...  │
│                                          │
│  [문의하기] [다운로드 샘플]              │
└─────────────────────────────────────────┘
```

**구현 요소**:
- [ ] 모달 컴포넌트 생성
- [ ] 갤러리 아이템 클릭 시 모달 열기
- [ ] 작품 상세 정보 표시
- [ ] "문의하기" 버튼 (현재는 toast 메시지)
- [ ] ESC 키 / 외부 클릭으로 닫기
